# Arada Sub-City Code Enforcement

A modern React application for the Arada Sub-City Code Enforcement Authority, designed to facilitate urban regulation compliance and citizen reporting in Addis Ababa.

## 🌐 Live Website
The application is live and accessible at:
### **[abeyeshitila81.github.io/arada-code-enforcement/](https://abeyeshitila81.github.io/arada-code-enforcement/)**

---

## ✨ Features
- **Citizen Reporting**: Easily report code violations (illegal construction, waste dumping, etc.).
- **Official Documentation**: Access public regulations and official forms and reports.
- **Latest News**: Stay updated with community announcements and policy changes.
- **Bilingual Support**: Important information provided in both English and Amharic.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop users.

## 🛠️ Technology Stack
- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Deployment**: GitHub Pages via GitHub Actions

## 📂 Project Structure
- `/frontend`: The main React application source code.
- `Ale app.html`: Original legacy HTML file for reference.
- `style.css`: Original legacy styles for reference.

## 🚀 Development
To run the project locally:
1. Clone the repository.
2. Navigate to the `frontend` directory.
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
